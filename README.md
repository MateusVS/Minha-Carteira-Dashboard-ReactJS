# Instalação

Execute os seguintes comandos em um terminal:

```
git clone https://github.com/MateusVS/Minha-Carteira-Dashboard-ReactJS.git
```

```
cd Minha-Carteira-Dashboard-ReactJS
```

```
yarn
```

# Para rodar o projeto

Utilize o comando:

```
yarn start
```
